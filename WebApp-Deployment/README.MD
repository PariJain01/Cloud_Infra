### 1Ô∏è‚É£ Create an App Service Plan

```bash

  --name myAppServicePlan \
  --resource-group myResourceGroup \
  --sku FREE \
  --is-linux
```

---

### 2Ô∏è‚É£ Create a Web App

```bash
az webapp create \
  --resource-group myResourceGroup \
  --plan WelcomePagePlan \
  --name WelcomePage \
  --runtime "NODE:18-lts" \
  --deployment-local-git
```

---

### 3Ô∏è‚É£ Enable Local Git Deployment
